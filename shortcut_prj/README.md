# ⚡ 쇼특허 (Short-Cut) - AI 특허 분석 서비스

**쇼특허(Short-Cut)**는 전 세계 특허 데이터를 실시간으로 조회하여 사용자의 아이디어를 진단하고, 유사도 및 침해 리스크를 분석해주는 AI 기반 웹 서비스입니다.

---

## 🛠 기술 스택 (Tech Stack)

| 구분 | 기술 | 설명 |
| :--- | :--- | :--- |
| **Framework** | **Django 5.0.0** | 파이썬 기반 고성능 웹 프레임워크 |
| **Language** | **Python** | 데이터 처리 및 백엔드 로직 구현 |
| **AI / Data** | **OpenAI, Pinecone** | LLM 기반 분석 및 벡터 데이터베이스 검색 |
| **Database** | **SQLite3** | 관계형 데이터베이스 관리 (사용자 및 세션 데이터) |
| **Frontend** | **HTML5, CSS3, JS** | 반응형 다크 모드 UI 및 실시간 애니메이션 |

---

## ✨ 핵심 기능 (Key Features)

### 1. 사용자 인증 시스템
* **회원가입 및 로그인**: Django의 `UserCreationForm`과 `auth_views`를 활용한 보안 인증.
* **세션 관리**: 사용자별 로그인 상태 유지 및 로그아웃 기능 지원.

### 2. AI 아이디어 분석
* **특허 유사성 점수**: 입력된 아이디어와 기존 특허 기술 간의 일치도 산출.
* **침해 위험도 분석**: 선행 특허권리 범위 포함 가능성을 '높음/낮음' 단계로 시각화.
* **전문가 회피 전략**: AI가 생성한 차별화 포인트 및 기술적 개선 제언 제공.

### 3. 직관적인 대시보드 UI
* **채팅 기반 인터페이스**: 챗봇 형태의 UI로 아이디어를 쉽고 빠르게 입력.
* **분석 기록 사이드바**: 최근 분석한 이력을 사이드바에서 즉시 확인 가능.
* **리포트 저장 기능**: 분석 결과를 확인하고 즉시 PDF 리포트로 인쇄 및 저장.
